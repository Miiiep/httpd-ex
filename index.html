Mein Wissen über die Cloud

  • Die Cloud bietet die Möglicht Daten online zu speichern
  • Mit Cloud Computing ist es möglich, Server, Clients, Datenspeicher und vieles mehr zu hosten
  • Die Cloud bietet viel Sicherheit, da Daten nicht mehr Lokal sondern dezentral gespeichert werden.
  • Daten in der Cloud sind von überal zugänglich. Sofern internet vorhanden ist.
  • Mit der Cloud können erhebliche kosten gespaart werden da meist nur genutze Ressourcen bezahlt werden müssen

Grundlagen Cloud-Computing

  • Wie ist der Begriff Cloud entstanden? Wieso heisst es Cloud?

    • Der Begriff "Cloud" entstand in Anlehnung an die Symbolisierung von Netzwerkdiagrammen, in denen eine Wolke verwendet wurde, um das Internet darzustellen. Als dann Cloud Computing aufkam wurde diese Symbolik übernommen


  • Wie wird der Begriff Cloud definiert, z.B. gemäss NIST?

    • Cloud Computing ist ein Modell, das den allgegenwärtigen, bequemen und bedarfsgesteuerten Netzwerkzugriff auf einen gemeinsamen Pool von konfigurierbaren Rechnerressourcen (z. B. Netzwerke, Server, Speicher, Anwendungen und Dienste) ermöglicht, die schnell bereitgestellt und freigegeben werden können mit minimalem Managementaufwand oder Service Provider-Interaktion. Dieses Cloud-Modell besteht aus fünf wesentlichen Charakteristika, drei Service-Modellen und vier Bereitstellungsmodellen.


  • Welches sind die 5 Merkmale einer Cloud?

    • On-Demand Self Service (Kann von User jederzeit angepasst werden)

    Broad Network Access (Fähigkeit sich mit einer Vielzahl von Geräten zu verbinden)

    Resource Pooling (viele User teilen viele Ressourcen)

    Rapid Elasticity/Schnelle Skalierbarkeit (Cloud-Services können sehr schnell aufgestellt werden)

    Measured Service(Pay per use)
